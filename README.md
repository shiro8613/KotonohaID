# KotonohaID (Fabric)

## 🌟 Modの概要

**KotonohaID**は、Minecraftのコマンド操作を大幅に改善するFabric Modです。

コマンドでアイテムIDを入力する際、デフォルトでは `minecraft:dirt` のような内部名を入力する必要がありますが、このModを導入することで、**アイテムの表示名（例: 「土」や「ダイヤモンドの剣」）からアイテムIDの入力補完を効かせられる**ようになります。これにより、コマンドの入力ミスを減らし、スムーズな操作を実現します。

  * **対応Minecraftバージョン:** fabric 1.21.10
  * **前提条件:** なし (Fabric Loaderのみが必要です)

## 🚀 主要な機能 (Features)

  * **表示名からのID補完:** `/give` や `/setblock` などのコマンドで、アイテムIDの引数部分に日本語（または設定された言語）のアイテム名を入力し始めると、対応する正式なアイテムID（例: `minecraft:diamond_sword`）が候補として表示されます。
  * **開発者向け支援:** 複雑なアイテムIDを覚える必要がなくなり、Mod開発やアドベンチャーマップ作成におけるコマンド記述の効率が向上します。

## 🛠️ 導入方法 (Installation Guide)

このModはクライアント側で動作します。(サーバー側は非対応です。)

1.  **Fabric環境の準備:**
      * お使いのMinecraftのバージョン（1.21.10）に対応した **Fabric Loader** をインストールしてください。
2.  **Modファイルのダウンロード:**
      * [リリースページ](https://github.com/shiro8613/KotonohaID/releases) にアクセスし、最新バージョンの `KotonohaID-x.x.x.jar` ファイルをダウンロードします。
3.  **ファイルの配置:**
      * ダウンロードした `.jar` ファイルを、Minecraftのインストールフォルダ内にある **`mods`** フォルダに配置します。
4.  **起動:**
      * Minecraftランチャーから「fabric-loader」のプロファイルを選択してゲームを起動します。
